Users speak a research task in natural language; Voyance watches its own screen, interprets every pixel with Gemini's multimodal vision, visits live websites, extracts structured data, and returns a spoken briefing with a downloadable comparison report.No DOM hacks. No site-specific APIs. Pure visual understanding — grounded by Perplexity, voiced by ElevenLabs.The name Voyance fuses Voyage (to navigate and explore) + Clairvoyance (to see what others cannot). The product travels the web with the perceptive clarity of a seasoned analyst — and delivers what it sees with a voice you can trust.Problem: Research is Manual &amp; Time-ConsumingSolution: Autonomous Visual Navigation + VoiceEdge: Gemini Vision + ElevenLabs Vera Persona02 Problem StatementOne-Sentence Problem Statement (PRD Format)Knowledge workers waste 30-40% of their workday manually browsing, copy-pasting, and cross-referencing web sources to produce research reports that an intelligent visual agent could generate in under 3 minutes.Pain Points in DetailTime sink: Analysts, founders, and consultants spend hours on competitor research that yields the same templated output every time.Context collapse: Switching between 10+ browser tabs destroys cognitive focus and increases error rates.API fragility: Existing research tools break whenever a target site changes its DOM — Voyance never breaks because it reads pixels, not code.No voice-directed workflow: Current browser automation tools require programming or rigid GUI builders — inaccessible to non-technical users.Hallucination risk: LLM-only research tools fabricate citations. USER speaks research brief via browser microphone.2. GEMINI LIVE API streams audio in real-time, transcribes, and extracts task parameters.3. ADK checks for VOICE INTERRUPTION via Gemini Live API. If received, agent replans.10. Loop repeats for all target sites (typically 3-7 per session).11. SYNTHESIS: ADK aggregates all data, generates final structured JSON report.12. ELEVENLABS VERA delivers spoken briefing.
